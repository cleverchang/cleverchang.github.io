数据结构与算法Java版——中缀表达式转后缀表达式

发表于 2017-10-18 | 分类于 数据结构与算法Java版 |

今天分享的是利用栈将中缀表达式转化为后缀表达式。

中缀表达式转后缀表达式的三个要点：

  • 1、数字直接输出
    • 2、括号匹配出栈，即括号凑成一对就从栈中弹出并输出。
    • 3、入栈运算符与栈顶元素发生比较，如果遇到的栈顶运算符一直小于入栈运算符优先级则不断弹出并输出，直到遇到更大优先级的运算符或栈为空。（* / 优先级大于 + - 优先级）
