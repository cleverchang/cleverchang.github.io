css基础小技巧——滑动框与固定位置

发表于 2017-05-25 | 分类于 CSS |

　　今天自己做学校选课网站前台页面时踩了两个坑，在此记录分享：

　　1、前台页面文字以及图片等的位置固定以及相对位置。

　　２、滑动框的显示与设置。

　　问题一：注意设置相对位置时要设置position：relative；并且最后用top,left设置位置，最好auto自适应。不一定需要设死。

　　问题二：注意有没有加入overflow: hidden;这句话，如果想要消除滑动框可以用这句，如果需要滑动框，则设置为overflow: auto;(div标签设置滑动框也可以如此设置)
