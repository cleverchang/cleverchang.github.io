　　contentType：用于指定发送给Web浏览器的流的MIME类型，默认为text/plain。

　　contentDisposition：用于设置响应的Content-Disposition的头，指定下载文件的名称：

　　　inline;filename=“文件名称”；inline表示当前页面打开下载文件（默认值）

　　　attachment;filename=“文件名称”；attachment表示打开一个“文件下载”对话框

　　inputName:用于指定action链中InputStream属性的名称，默认为inputStream
