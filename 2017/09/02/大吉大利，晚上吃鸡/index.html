大吉大利，晚上吃鸡

发表于 2017-09-02 | 分类于 感想与生活 |

不知道你有没有突然发现身边总有人喊着：“今晚吃鸡，吃鸡balabala的”。甚至都有了以下表情包：

吃鸡到底是个什么梗？大家不用想歪，这个鸡并不是那个鸡，吃鸡 slogan 来源于全球火爆的一款游戏——绝地求生，这是一款求生 + 射击 + 团队 + balabala 的游戏，而“大吉大利，晚上吃鸡”正是拿了第一名后屏幕显示的，英文原文：“Winner Winner ，Chicken dinner”。现在打开任何一个直播平台，你一定可以看见各大主播都会玩这个。昨晚熊猫TV托马斯和林更新等人的直播引来了160W人的观看，听说王思聪很久没露面就是因为沉迷其中，无法自拔。

为什么绝地求生会这么火？

仔细思考，发现这款游戏有火的必然性：

  • 现在市场上的供人们玩耍的网络游戏并不多，LOL（英雄联盟）火了7年之久，很多人开始厌倦了这个模式，而老牌的射击游戏CF（穿越火线），CS（反恐精英）操作难度高，高手与新手之间的差距可以让人怀疑人生，不适合广大群体，其他游戏亦然。
  • 绝地求生这款游戏画质极好，操作性强，逼真的模仿现实，所有武器随机刷，只要跳伞位置好，抢到好枪打大佬。一个小白都可以一开场就干掉别人，或者苟活做个伏地魔，反正怎么开心怎么玩。
  • 游戏本身就具备观赏趣味性，而各大平台主播的直播起到了推波助澜的作用。
  • ……

游戏确实火了，不过这个游戏能火多久？会不会代替其他游戏，出现像LOL占据网吧的现象？

个人认为目前不会，这个游戏门槛相比其他游戏还是高了点，买这款游戏 98 元，外加加速器付费，所以费用会高出许多。不要小看了这些钱，对于小学生，初中生和大部分高中生还是一笔不小的费用，这注定了绝地求生的受众相对较少，不像LOL老少皆玩。

其次，游戏虽然小白可以上手，但是想活到最后几乎是不可能，每天打了这么久，吃不到一把鸡，耐心一般都会磨灭。况且这款游戏摇晃厉害，对于以前没玩过这种类型的人，可能会产生头晕不适。

个人也玩了两把，发现如果你运气足够“好”，一直没碰到人，那你游戏体验就无法描述了，是幸运苟活下来了？还是不幸竟然一个人都碰不到。

最最重要的是，妹子玩射击类游戏的少，广大妹子并不见得青睐这款游戏。（想在这游戏里面找女朋友还不如去农药）

不过听说腾讯会代理这款游戏，如果真这么做，游戏速度肯定加快，每个月就节省了 30 块的加速器费用，那个时候确实存在普及这个游戏的可能，并且还不用申请账号，直接QQ登录，然后各种虐国服小学生的视频就出来了……

不过“吃鸡”的火爆也直接导致了很多“夜猫”的诞生，前几日某主播，不吃到鸡不停播，结果……第二天了。前两天室友也是通宵达旦，根据他自己描述：“我嘴巴上说要不休息会吧，结果还一直再捡枪……”，看来嘴巴上说不要，身体还是很诚实。所以还是要温馨提示：“吃鸡”乃长久之事，不要太过频繁。毕竟来日方长。
